AI Security Engineer focused on the intersection of application security and large language models. I build tooling to detect and remediate vulnerabilities — spanning static analysis, AI-generated code risks, and autonomous remediation pipelines.

## Featured Work

### [aiscan](https://github.com/samsepassi1/aiscan) — Hybrid AST + LLM Security Scanner

A static analysis tool that combines traditional AST parsing with LLM-powered reasoning to detect security vulnerabilities in AI-generated code. Covers prompt injection, hardcoded secrets, unsafe deserialization, path traversal, and more across code produced by Copilot, Cursor, ChatGPT, and Claude.
